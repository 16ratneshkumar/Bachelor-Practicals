# DSC-16: Artificial Intelligence - Practicals

![Course](https://img.shields.io/badge/Course-DSC--16-blue.svg?style=for-the-badge)

Welcome to the collection of practical assignments for the **Artificial Intelligence** course. This repository contains implementations for various logic-based problems and algorithms using **Prolog**.

## üõ†Ô∏è Environment Setup

This project uses **Prolog**. We recommend using **SWI-Prolog** for running these programs.

**Quick Setup:**
1.  **Install SWI-Prolog:** Download and install from [swi-prolog.org](https://www.swi-prolog.org/).
2.  **Load the File:** Open the terminal in this directory and run:
    ```prolog
    ?- [practical].
    ```
    *(Note: Ensure you are loading the `.pl` file)*
3.  **Run Queries:** After loading, you can start querying predicates like `father(john, X).` or `factorial(5, F).`

## üìÖ List of Practicals

This repository covers the following 15 practicals as implemented in [Practical.pl](./Practical.pl):

1.  **[Family Tree Implementation](./Practical.md#1-family-tree-implementation)** - Define family relationships using facts and rules.
2.  **[List Concatenation](./Practical.md#2-list-concatenation)** - Concatenate two lists into a third list.
3.  **[List Reversal](./Practical.md#3-list-reversal)** - Reverse the elements of a list.
4.  **[Sum of Two Numbers](./Practical.md#4-sum-of-two-numbers)** - Calculate the sum of two numerical values.
5.  **[Maximum of Two Numbers](./Practical.md#5-maximum-of-two-numbers)** - Find the larger of two numbers.
6.  **[Factorial](./Practical.md#6-factorial)** - Compute the factorial of a given number.
7.  **[Fibonacci Series](./Practical.md#7-fibonacci-series)** - Generate the Nth Fibonacci number.
8.  **[Power Function](./Practical.md#8-power-function)** - Calculate the power (Num^Pow).
9.  **[Multiplication](./Practical.md#9-multiplication)** - Implement multiplication through recursive addition.
10. **[Membership Check](./Practical.md#10-membership-check)** - Check if an element is a member of a list.
11. **[Sum of List](./Practical.md#11-sum-of-list)** - Calculate the total sum of elements in a list.
12. **[Even/Odd Length](./Practical.md#12-evenodd-length)** - Determine if the length of a list is even or odd.
13. **[Maximum in List](./Practical.md#13-maximum-in-list)** - Find the maximum value within a list.
14. **[Insert at Position](./Practical.md#14-insert-at-position)** - Insert an item at a specific index in a list.
15. **[Delete at Position](./Practical.md#15-delete-at-position)** - Remove an element from a specific index in a list.
